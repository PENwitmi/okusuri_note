抗血栓薬 完全マスターガイド

血を固まらせない薬の使い分け

「動脈には抗血小板薬、静脈には抗凝固薬」－この基本原則の理由を理解すれば、複雑に見える抗血栓薬の選択が明確になります。白色血栓と赤色血栓の違いから、最新のDOACまで、抗血栓療法の全体像をマスターしましょう。

🤔 なぜ抗血小板薬と抗凝固薬の2種類が必要？

抗血栓薬を理解する鍵は、血栓の「色」にあります。動脈で形成される白色血栓と、静脈で形成される赤色血栓では、形成メカニズムが全く異なるため、それぞれに特化した薬剤が必要なのです。

⚪

白色血栓（動脈血栓）

形成場所：動脈（高速血流）

主成分：血小板

トリガー：血管内皮障害

疾患：心筋梗塞、脳梗塞

→ 抗血小板薬で対応
🔴

赤色血栓（静脈血栓）

形成場所：静脈（低速血流）

主成分：赤血球＋フィブリン

トリガー：血流うっ滞

疾患：深部静脈血栓症、肺塞栓症

→ 抗凝固薬で対応

覚えておきたい基本原則

  • 高速血流（動脈）では血小板が活性化しやすい → 抗血小板薬
  • 低速血流（静脈）では凝固因子が蓄積しやすい → 抗凝固薬
  • 心房細動は心房内の血流うっ滞 → 抗凝固薬（静脈血栓型）
  • ステント留置後は血小板が活性化 → 抗血小板薬（DAPT）

🎯 薬剤はどこに効く？

抗血小板薬の作用点

血小板が活性化する主な経路は2つ：

1. TXA2経路

血小板内でアラキドン酸からTXA2（トロンボキサンA2）が作られる

→ アスピリンがこの生成をブロック

2. ADP経路

血小板が放出するADPがP2Y12受容体に結合

→ クロピドグレル等がこの受容体をブロック

抗凝固薬の作用点

凝固カスケードの重要ポイントをブロック：

多くの凝固因子
↓
第Xa因子
リバーロキサバン、アピキサバン、エドキサバンがブロック
↓
トロンビン
ダビガトランがブロック
↓
フィブリン（血栓の網）

ワルファリンは少し違う働き方：

ビタミンKを邪魔して、凝固因子の産生自体を減らす

❓ 薬学生のよくある疑問Q&A

実習や国試でよく疑問に思うことをQ&A形式でまとめました！
基礎知識は上の表までで、ここからが特集の本編です。

Q1: DOACが便利なのに、ワルファリンを使うことってあるの？

A: はい、以下の場合はワルファリンしか使えません！

🚫 DOACが使えない4つのケース
  1. 機械弁（人工弁）

    DOACでは血栓・出血リスクが増加（RE-ALIGN試験）

  2. 重度腎不全（CrCl＜15）

    DOACは腎排泄のため使用不可

  3. 妊娠中

    DOACの安全性データ不足

  4. 抗リン脂質抗体症候群

    ワルファリンの方が効果的

💡 覚え方：「キニジン」（機械弁・腎・妊娠・抗リン脂質）

Q2: 抗血小板薬と抗凝固薬を併用することってあるの？

A: はい、実はよくあります！

1️⃣ 抗血小板薬同士の併用：DAPT（ダプト）

💊💊 アスピリン + クロピドグレル（または他のP2Y12阻害薬）

異なる経路で血小板をブロック → より強力な予防効果

いつ使う？

  • ステント留置後（PCI後）
  • 心筋梗塞後
  • 期間：通常3-12ヶ月
2️⃣ 抗血小板薬＋抗凝固薬の併用

PCI後＋心房細動の患者さん

🤔 なぜ併用？

  • ステント内血栓（白色血栓）→ 抗血小板薬が必要
  • 心房内血栓（赤色血栓）→ 抗凝固薬が必要

⚠️ ただし出血リスクが上がるので、期間を最小限に！

Q3: CHADS₂-VAScスコアが覚えられません

A: まず「年齢」と「脳梗塞既往」だけ覚えればOK！

🎯 実習で使える簡易判断法
  1. 75歳以上 → 2点（これだけで抗凝固薬開始！）
  2. 脳梗塞既往 → 2点（これも即開始！）
  3. 65-74歳 → 1点（もう1つリスクあれば開始）

この3つで判断の8割はカバーできます！

💡 その他の項目（各1点）

C：心不全、H：高血圧、D：糖尿病、V：血管疾患、Sc：女性

Q5: プラスグレル20mgって多すぎない？

A: それは初回負荷投与（Loading dose）です！

Loading doseとは？

いつ使う？：急性冠症候群（ACS）やPCI時など、緊急で血小板阻害が必要な時

なぜ必要？：通常量では効果発現まで数日かかるため、初回に大量投与してすぐに治療域に到達させる

タイミング：PCIの前または直後、診断確定後できるだけ早く

プラスグレル（エフィエント）の用量

  • 初回：20mg（Loading dose）
  • 維持量：3.75mg/日

※ 5倍以上の差があるのは、急速に血小板機能を抑制するため

他の薬剤のLoading dose

  • クロピドグレル：300-600mg → 75mg/日
  • チカグレロル：180mg → 90mg×2/日

その他よくある疑問

Q6: リバーロキサバンだけなぜ食後？

A: 15mg、20mg錠は食事で吸収が改善されるため

  • 空腹時：バイオアベイラビリティ66%に低下
  • 食後：100%の吸収
  • ※ 10mg錠は食事の影響なし

Q7: クロピドグレルとPPIの相互作用って本当？

A: はい、CYP2C19を介した重要な相互作用があります

メカニズム

クロピドグレルはプロドラッグで、CYP2C19により活性代謝物に変換されて効果を発揮します。

一部のPPIがCYP2C19を阻害 → 活性代謝物が減少 → 血小板阻害効果が最大70%減弱

❌ 避けるべきPPI
  • オメプラゾール（オメプラール）
  • エソメプラゾール（ネキシウム）

強力なCYP2C19阻害作用

✓ 推奨されるPPI
  • ランソプラゾール（タケプロン）
  • ラベプラゾール（パリエット）

CYP2C19阻害作用が弱い

💡 実践ポイント：DAPT中は消化管出血予防のPPIが必須。必ずランソプラゾールかラベプラゾールを選択！

Q8: 周術期の中止期間が覚えられません

A: 「24の法則」で覚えよう！

DOAC：基本24時間前、リスクが上がるごとに+24時間

抗血小板薬：

  • アスピリン・プラスグレル：7日前
  • クロピドグレル：5日前
  • チカグレロル：3日前（可逆的だから短い）
