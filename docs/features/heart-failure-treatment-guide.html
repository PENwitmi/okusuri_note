心不全治療の新時代

2025年ガイドライン改訂とHFrEF/HFpEF治療戦略

7年ぶりの全面改訂となった2025年心不全ガイドライン。HFrEFの「Fantastic Four」による予後改善と、「治療法がない」と言われたHFpEFへの新たな治療選択肢。心不全治療のパラダイムシフトを包括的に解説する。

なぜ今、心不全治療が大きく変わったのか？

心不全は「心臓のポンプ機能の障害により、生体の需要に見合った血液量を拍出できない状態」。しかし、その治療戦略は2010年代後半から劇的に進化した。特にHFpEFへの治療薬の登場は、心不全診療に革命をもたらしている。

2025年ガイドライン改訂の3つのポイント

  • LVEF分類の国際標準化：HFrEF（≤40%）、HFmrEF（41-49%）、HFpEF（≥50%）への統一
  • HFrEF治療の進化：「Fantastic Four」の早期同時導入による予後改善
  • HFpEF治療の確立：SGLT2阻害薬、フィネレノン、GLP-1受容体作動薬の推奨

心不全の新分類（2025年基準）

HFrEF LVEF ≤40% 収縮不全 4本柱治療
→
HFmrEF LVEF 41-49% 軽度収縮不全 個別化治療
→
HFpEF LVEF ≥50% 拡張不全 新規治療

HFpEF：なぜ収縮が正常なのに心不全になるのか？

HFpEF（Heart Failure with preserved Ejection Fraction）は、左室収縮能が保たれているにも関わらず心不全症状を呈する病態。長年「拡張不全」と呼ばれてきたが、その本質はより複雑である。

HFpEFの重要な事実

  • 心不全全体の約50%を占める（高齢化により増加傾向）
  • 予後はHFrEFと同等に不良（5年生存率約50%）
  • 2020年まで有効な治療薬が存在しなかった

HFpEFの多面的な病態

1. 拡張障害（コアメカニズム）

  • 心筋スティフネス増加
  • 能動的弛緩障害
  • 左房圧上昇
  • 肺うっ血

2. 全身性炎症

  • 併存疾患による慢性炎症
  • 酸化ストレス増加
  • 内皮機能障害
  • 微小血管障害

3. 代謝異常

  • 肥満（BMI ≥30が50%）
  • 糖尿病（30-40%）
  • インスリン抵抗性
  • 脂質異常症

4. 腎機能障害

  • 心腎連関
  • 体液貯留
  • 電解質異常
  • 貧血

HFpEF発症の流れ

加齢・高血圧・糖尿病・肥満
→
全身性炎症・酸化ストレス
→
心筋線維化・血管内皮障害
→
拡張機能障害・左房圧上昇
→
HFpEF（労作時呼吸困難・運動耐容能低下）

HFpEF治療革命：「治療法がない」から「治療できる」へ

2020年代に入り、HFpEFに対する有効な治療薬が続々と登場。2025年ガイドラインでは、患者背景に応じた個別化治療が推奨されている。

主要薬剤の詳細解説

ARNI（サクビトリルバルサルタン）：HFrEF治療の新基準

基本情報

  • 商品名：エンレスト錠
  • 作用機序：ネプリライシン阻害＋ARB
  • 適応：慢性心不全（HFrEF）
  • 特徴：ACE阻害薬を上回る予後改善効果

PARADIGM-HF試験（2014年）

エナラプリルとの比較で：

  • 心血管死：20%減少（HR 0.80）
  • 心不全入院：21%減少
  • 全死亡：16%減少
  • NNT：21（心血管死予防）

実践的ポイント

  • ACE阻害薬から切り替え時は36時間のウォッシュアウト必須
  • 血管浮腫のリスクに注意（ACE阻害薬既往例は慎重投与）
  • 低血圧・高K血症・腎機能に注意しながら増量
  • 咳嗽の副作用はACE阻害薬より少ない

SGLT2阻害薬：HFrEF・HFpEF両方に有効な万能薬

基本情報

  • 代表薬：エンパグリフロジン（ジャディアンス）、ダパグリフロジン（フォシーガ）
  • 作用機序：近位尿細管でのグルコース再吸収阻害＋心腎保護作用
  • 適応：慢性心不全（HFrEF・HFpEF）、CKD、2型糖尿病
  • 特徴：糖尿病の有無に関わらず効果

主要臨床試験

HFrEF

DAPA-HF：心血管死・心不全悪化26%減少

EMPEROR-Reduced：同25%減少

HFpEF

DELIVER：心血管死・心不全悪化18%減少

なぜ心不全に効くのか？

  • 利尿作用：浸透圧利尿による前負荷軽減
  • 代謝改善：ケトン体産生による心筋エネルギー効率改善
  • 抗炎症作用：心筋線維化抑制
  • 血管機能改善：動脈スティフネス改善
  • 交感神経抑制：心拍数減少効果

使用上の注意

  • eGFR 20mL/min/1.73m²以上で開始可能
  • 初期に一過性のeGFR低下（dip）あり→継続により回復
  • 尿路・性器感染症に注意（特に女性）
  • 正常血糖ケトアシドーシス（まれ）
  • 脱水・血圧低下に注意（利尿薬併用時）

フィネレノン：第4世代MR拮抗薬のHFpEF治療

基本情報

  • 商品名：ケレンディア錠

GLP-1受容体作動薬：肥満合併HFpEFの新治療

基本情報

  • 代表薬：セマグルチド（オゼンピック、ウゴービ）
  • 作用機序：GLP-1受容体刺激による多面的効果
  • 適応：2型糖尿病、肥満症
  • 特徴：体重減少を介したHFpEF改善

STEP-HFpEF試験（2023年）

肥満（BMI≥30）合併HFpEFでの画期的成果：

  • KCCQ-CSS：+7.8ポイント改善
  • 体重減少：13.3%（vs 2.6%）
  • 6分間歩行距離：+20.3m
  • CRP：43%減少
  • 心血管死・心不全イベント：31%減少（プール解析）

肥満とHFpEFの悪循環を断つ

悪循環：肥満 → 全身性炎症 → 心臓周囲脂肪 → 拡張障害 → 運動制限 → さらなる肥満

↓ GLP-1受容体作動薬

好循環：体重減少 → 炎症軽減 → 心臓負荷軽減 → 運動耐容能向上 → QOL改善

実践的使用法

  • 対象：BMI ≥30のHFpEF患者
  • 週1回皮下注射（セマグルチド）
  • 用量漸増：0.25mg → 2.4mg（16週間かけて）
  • 消化器症状（悪心・嘔吐）は継続により軽減
  • SGLT2阻害薬との併用推奨

よくある落とし穴と対策

  • 「血圧が低いから薬を増やせない」
    → 症状がなければ収縮期90mmHg以上で継続可能。長期予後改善が優先
  • 「腎機能が悪化したから中止」
    → 30%以内の上昇は想定内。長期的には腎保護効果
  • 「高齢だから標準量は無理」
    → 年齢より忍容性で判断。可能な限り目標用量を目指す
  • 「症状が改善したから減量・中止」
    → 予後改善薬は症状に関わらず継続が原則

まとめ：心不全治療の新時代

Key Messages

  1. HFrEFは「Fantastic Four」で予後が劇的に改善
    4剤併用により死亡リスクを50%以上減少可能
  2. HFpEFは「治療できる病気」になった
    SGLT2阻害薬を基本に、併存疾患に応じた個別化治療
  3. 早期診断・早期治療介入が重要
    進行してからでは改善困難。初期からの積極的治療を
  4. 薬物治療は「足し算」の考え方
    各薬剤が異なる機序で相乗効果を発揮

今後の展望

  • 新規薬剤の開発：オミカムチブ（心筋ミオシン活性化薬）、ベリシグアト（sGC刺激薬）
  • 個別化医療の進展：バイオマーカーによる治療選択
  • デバイス治療との統合：薬物＋CRT/ICDの最適化
  • 予防医療へのシフト：Stage A/Bでの介入強化

Clinical Pearl

「心不全治療は marathon, not a sprint」
一つ一つの薬剤を忍容性を見ながら確実に導入し、長期的な予後改善を目指す。2025年の今、我々は心不全と戦う強力な武器を手にしている。それを最大限に活用することが、患者の生命予後とQOL改善につながる。
