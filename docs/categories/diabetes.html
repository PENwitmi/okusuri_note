患者背景を選択してください：

🫀 心血管疾患既往患者

第一選択：エンパグリフロジン（ジャディアンス®）

選択理由：

  • EMPA-REG OUTCOME試験：心血管死38%減少
  • 心筋梗塞二次予防での明確なエビデンス
  • 循環器内科医の圧倒的支持

用法用量：10mg 1日1回朝食前 → 25mg（効果により増量）

モニタリング：月1回のHbA1c、eGFR、血圧測定

💚 心不全合併患者

第一選択：ダパグリフロジン（フォシーガ®）

選択理由：

  • DAPA-HF試験：HFrEFで心不全入院26%減少
  • DELIVER試験：HFpEFでも有効性確認
  • 糖尿病の有無を問わない適応症

用法用量：10mg 1日1回朝食前（糖尿病の有無問わず）

併用推奨：ACE阻害薬/ARB、β遮断薬、MRAとの4剤併用

🟣 CKD合併患者

第一選択：ダパグリフロジン（フォシーガ®）

選択理由：

  • 唯一のCKD適応（eGFR 25-75で使用可能）
  • DAPA-CKD試験：eGFR低下44%抑制
  • 非糖尿病性CKDでも保険適応

用法用量：10mg 1日1回（eGFR≥25で継続可能）

特記事項：eGFR低下は一時的に許容（長期的には保護的）

👴 高齢者（75歳以上）

第一選択：トホグリフロジン（デベルザ®）

選択理由：

  • 日本人高齢者での豊富な安全性データ
  • 低用量から開始可能（20mg）
  • 脱水リスクが相対的に低い

用法用量：20mg 1日1回朝食後（通常用量の半量から）

注意事項：脱水症状、起立性低血圧の定期確認

⚖️ 肥満合併糖尿病

第一選択：エンパグリフロジン（ジャディアンス®）25mg

選択理由：

  • 300-400kcal/日のカロリー喪失効果
  • 体重減少効果：2-4kg期待
  • 内臓脂肪減少による代謝改善

用法用量：10mg → 25mg（体重減少効果増強のため）

併用効果：GLP-1受容体作動薬との相乗効果期待

📋 標準的2型糖尿病

第一選択：エンパグリフロジンまたはダパグリフロジン

選択理由：

  • 将来の心血管・腎イベント予防効果
  • 低血糖リスクが極めて低い
  • 体重減少・血圧降下の副次効果

判断基準：迷ったらエンパグリフロジン（EMPA-REGの信頼性）

長期目標：HbA1c改善と同時に臓器保護を実現
