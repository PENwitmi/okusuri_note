アトルバスタチン

アトルバスタチン （リピトール®）
HMG-CoA還元酵素阻害薬（スタチン系） 史上最も売れた薬剤の伝説 - スタチン戦争の絶対王者
発売年 1997年
累計売上 1,640億ドル
LDL低下率 最大60%
市場シェア 最盛期60%

「史上最も売れた薬剤の伝説」

1997年の登場以来、アトルバスタチンは単一薬剤として人類史上最高の 1,640億ドルの売上記録を打ち立てました。「リピトール」として 15年間世界最大の処方薬の地位を独占し、スタチン戦争と呼ばれた激烈な競争を制覇。 その成功は薬学史の金字塔として、現代製薬業界の 「完璧な成功モデル」となっています。

スタチン戦争と伝説の誕生

1990年代 スタチン戦争の激化

1980年代末、プラバスタチンとシンバスタチンが市場を分割していた。 しかし、効果は確実だが限定的で、より強力なスタチンへの 医療現場のニーズが急速に高まっていた。

「史上最強のスタチンを作る」

ファイザーの研究開発チームによる挑戦が、 後に医薬品史上最高の成功をもたらすことになるとは、 この時誰も想像していなかった。

日本における特別な位置づけ

日本の脂質異常症治療における革命

「迷ったらアトルバスタチン」文化の確立

日本の循環器専門医の間で「最も信頼できるスタチン」として絶対的地位を確保。 15年間の圧倒的な使用実績により、処方時の第一選択薬として完全に定着している。

日本人の低用量有効性

欧米と比較して日本人では低用量（10-20mg）でも十分な効果を示すことが確認されている。 これにより副作用リスクを最小限に抑えながら、目標LDL値到達を実現。 日本人の体質に最適化された治療が可能となった。

ジェネリック移行後の継続信頼

2011年の特許切れ後、薬価が95%下落してもなお「ジェネリック・アトルバスタチン」として 処方継続される臨床的価値を確立。医師・患者双方の信頼が価格を超越している。
