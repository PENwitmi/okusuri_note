レベル1：薬学生向け基本情報

カルベジロール

カルベジロール （アーチスト®）
第3世代αβ遮断薬 α1遮断＋β遮断の二重遮断革命
発売年 1993年
COMET試験 死亡率17%減
抗酸化作用 +25%血管保護
投与回数 1日2回

基本情報

薬効分類

αβ遮断薬（非選択的β遮断＋α1遮断）

特徴

心不全に禁忌から必須薬への転換

適応症

心不全、高血圧、狭心症

作用機序

二重遮断メカニズム

β遮断作用：心拍数・収縮力低下→心筋酸素消費量減少

α1遮断作用：血管拡張→後負荷軽減

抗酸化作用：活性酸素除去→心筋保護

使い分けポイント

カルベジロールを選ぶ理由

  • 心不全（特に収縮機能低下）
  • 高血圧合併の心不全
  • 抗酸化作用が欲しい場合

ビソプロロールとの違い

  • カルベジロール：α1遮断あり、血管拡張効果
  • ビソプロロール：β1選択的、喘息により安全

レベル2：実習中向け詳細情報

薬物動態

吸収・分布

経口バイオアベイラビリティ：25-35%（肝初回通過効果大）

食事と一緒に服用→吸収改善

代謝・排泄

主にCYP2D6、CYP2C9で代謝

半減期：6-10時間

用法用量

心不全の場合

開始用量：1.25mg 1日2回

維持用量：10-20mg 1日2回

注意：2週間ごとに倍量ずつ増量

副作用と注意点

主な副作用

  • めまい、ふらつき（α1遮断による）
  • 徐脈、低血圧
  • 体重増加

禁忌・慎重投与

  • 急性心不全の急性期
  • 重篤な房室ブロック
  • 気管支喘息

レベル3：研修中向け高度情報

COMET試験の衝撃

試験概要

対象：慢性心不全患者3,029例

比較：カルベジロール vs メトプロロール

結果：死亡率17%減少（HR 0.83, p=0.0017）

なぜカルベジロールが勝ったのか

「α1遮断による血管拡張と抗酸化作用が、単純なβ遮断を上回った瞬間でした。心不全治療における新たなパラダイムの誕生です。」

循環器内科医師

臨床での実際の使い方

処方パターン例

心不全初回処方

アーチスト錠1.25mg 2錠 1日2回 朝夕食後

※2週間後に効果・副作用確認後、増量検討

薬剤師の視点

患者指導のポイント

  • 食事と一緒に服用（吸収改善）
  • 急な中止は避ける（リバウンド現象）
  • 体重増加時は医師に相談
  • めまい・ふらつきに注意
