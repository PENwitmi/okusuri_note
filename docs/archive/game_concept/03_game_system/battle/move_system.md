## PP（使用回数）システム

### PP消費量
- 頻回投与薬: PP多い（1日3-4回）
- 持続型薬剤: PP少ない（1日1回）
- 週1回製剤: PP極少（DPP-4阻害薬週1回）

### PP回復
- **製剤改良**: 徐放化でPP効率UP
- **剤型変更**: 注射→経口でPP回復簡単に
- **合剤化**: 複数効果で1回投与
