🏆 バンコマイシン - 国試出題第1位（30回）
💡 覚え方：「バンコマイシンはマッサ（MRSA）に効く」
⚠️ 最重要：TDM必須薬物（国試頻出No.1）
💉 TDM（血中濃度モニタリング）- 最頻出！
項目 数値 覚え方
有効トラフ値 10-20 μg/mL 「トー（10）ニー（20）」
中毒域 >20 μg/mL 「20超えたら毒」
半減期 6時間 「ロク（6）でも覚える」
📊 採血タイミング：次回投与直前（トラフ値測定）
【腎機能低下時の半減期】 CLcr 50mL/min → 半減期 12時間（2倍） CLcr 10mL/min → 半減期 60時間（10倍）
⚠️ 3大副作用 - 必ず出題される！
腎・赤・耳
副作用 特徴 対策
腎毒性 アミノグリコシド併用で増強 TDM実施、水分補給
レッドマン症候群 急速投与で発生 60分以上かけて投与
聴器毒性 高濃度・長期投与 聴力検査
💡 語呂：「ジン（腎）は赤い耳」で3大副作用
💊 適応・特徴 - 投与経路がカギ！
投与経路 適応 吸収
静注 MRSA感染症全般 －
経口 偽膜性腸炎のみ 吸収されない
⚠️ 超重要：経口投与は腸管内感染のみ有効
理由：消化管から吸収されないため全身感染には無効
✅ グラム陽性菌専用：グラム陰性菌には無効
📝 国試での問われ方TOP5
パターン1：TDM関連
「トラフ値10-20μg/mLを維持すべき薬物は？」
「半減期が6時間の抗MRSA薬は？」
パターン2：副作用
「急速投与により顔面紅潮を生じる抗菌薬は？」
「アミノグリコシドと併用で腎毒性が増強する薬物は？」
パターン3：適応
「経口投与で偽膜性腸炎に用いる薬物は？」
「MRSAによる肺炎の第一選択薬は？」
パターン4：投与設計
「腎機能低下時に投与間隔延長が必要な薬物は？」
「CLcr 30mL/minの患者への投与間隔は？」→24-48時間
パターン5：相互作用
「ループ利尿薬と併用で聴器毒性が増強する薬物は？」
「筋弛緩薬の作用を増強する抗菌薬は？」
🧮 計算問題対策（必出！）
【基本投与量】 通常：15-20 mg/kg × 1日2回 【投与間隔の調整】 CLcr > 80 → 12時間ごと CLcr 50-80 → 24時間ごと CLcr 10-50 → 48時間ごと CLcr < 10 → 週2-3回
例題：体重60kg、CLcr 30mL/minの患者
初回投与量：60kg × 20mg/kg = 1200mg
投与間隔：CLcr 30なので48時間ごと
⏱️ 30秒クイックレビュー（試験直前用）
絶対暗記リスト
①トラフ値：10-20 μg/mL
②半減期：6時間（腎正常時）
③副作用：腎・赤・耳
④レッドマン：60分以上で投与
⑤経口：偽膜性腸炎のみ
⑥併用注意：アミノグリコシド（腎毒性）
⑦採血：次回投与直前
