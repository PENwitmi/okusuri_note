## ゲーム内イベント

### 「耐性菌レイドバトル」
- MRSA、VRE、多剤耐性緑膿菌が巨大ボスとして登場
- 協力プレイで適切な抗菌薬を選択して撃破
- 報酬：レア抗菌薬、耐性克服アイテム

### 「抗菌薬適正使用キャンペーン」
- 広域抗菌薬の乱用でペナルティ
- 狭域抗菌薬での治療成功でボーナス
- AMR（薬剤耐性）対策を学習

## 教育的ポイント
- スペクトラムの概念（狭域→広域）
- 世代進化による改良点の理解
- 耐性機序と対策の学習
- 副作用プロファイルの把握
- PK/PD理論の基礎
