# CLAUDE.md - PharmaDex プロジェクト

## プロジェクト概要
**PharmaDex（ファーマデックス）- 未来薬局展示企画**

2025年夏の薬局チェーン向け未来型展示企画。薬学・医療の歴史から現在、そして未来への展望を、インタラクティブな展示で表現する大規模プロジェクト。

### コンセプト
「薬局の過去・現在・未来を一望できる体験型展示」
- 医薬品開発の歴史的マイルストーン
- 現代の課題（薬剤耐性、個別化医療など）
- 未来の薬局ビジョン（AI診断、遠隔医療、ナノ医療など）

## 特別ルール（重要）

### CEO×Manager協働体制（2025-06-27強化）
**単なるメッセージ転送ではなく、深い思考による価値創造を**

#### なぜ協働思考が必要か
1. **表面的な指示では教育的価値が生まれない**
   - ❌「薬の使い分けを書いて」→ 添付文書の羅列
   - ✅「なぜこの薬が必要になったか」→ 本質的理解

2. **PharmaDexの核心価値を実現するため**
   - 「なぜ似た薬が複数存在するのか」への深い回答
   - 感動的なストーリーと実用的知識の最適な融合
   - 学生から臨床医まで各レベルでの価値創造

#### 協働思考の実践方法
- **薬効群モデル設計時**：
  - CEO「この薬群で最も重要な臨床的価値は？」
  - Manager「現場では○○で悩むので、△△を重点的に」
  - → 共に価値の方向性を定めてから開発者へ指示

- **ストーリー統合時**：
  - CEO「感動と実用性のバランスは？」
  - Manager「ストーリー入口から詳細への導線を○○に」
  - → 十分な検討後に具体的な構成を決定

運用方法：
- CEO：展示の大局的ビジョン、本質的な問いかけ
- Manager：技術的実現性、実践的価値の具体化
- 両者の深い対話により、AIの真の価値を発揮

#### 思考と実行のバランス（重要）
**⚠️ 避けるべき2つの極端**
1. **思考不足**：即座にタスク分割 → 表面的な成果物
2. **思考の沼**：延々と議論 → 成果物が生まれない

**✅ 理想的なワークフロー**
1. **協働思考フェーズ**（30-60分）
   - 本質的な価値を探る
   - 「なぜ」を明確化
   - 伝え方を設計

2. **実行判断基準**（3つのチェック）
   - □ 具体的な指示が書けるか
   - □ 成果物が明確か
   - □ 品質基準が定まったか
   → すべてYESで実行へ

3. **開発展開**
   - 明確な指示に変換
   - 創造性も確保
   - 納期設定と管理

**📊 実践的なタイムボックス**
```
0-30分：CEO×Manager協働思考
  - 薬効群の本質的価値を探る

以降：進捗管理と品質確保
```
