# 吸入薬完全マスター特集ページ構成設計

## ページ全体構成
### タイトル
**吸入薬完全マスター - デバイス特性と適応別選択**

### リード文
吸入薬の選択と指導は薬剤師の重要な役割。テリルジー・レルベア・シムビコート・メプチンの4剤を徹底比較し、デバイス特性・適応症・吸入手技をマスターする。

## メインコンテンツ構成

### 1. イントロダクション（5%）
- 吸入薬の重要性と課題
- 本特集で学べること
- 対象薬剤の紹介（4剤）

### 2. 薬剤別詳細解説（40%）

#### 2.1 テリルジー（トリプル製剤の革命）
- **基本情報カード**
  - 成分：ICS/LAMA/LABA（世界初トリプル）
  - 適応：COPD（100のみ）、喘息（100/200）
  - なぜCOPDは100のみ？（肺炎リスクの観点）
- **エリプタデバイスの特徴**
  - 1日1回の簡便性
  - 操作方法（カバーを開けるだけ）
  - 吸入確認窓の見方
- **処方される患者像**
  - 重症COPD（GOLD グループE）
  - 高用量ICS/LABAでコントロール不良の喘息

#### 2.2 レルベア（年齢別の選択肢）
- **基本情報カード**
  - 成分：ICS/LABA
  - 適応：喘息（50/100/200）、COPD（100のみ）
  - 年齢別規格の意義（小児から成人まで）
- **エリプタデバイス**（テリルジーと共通）
- **処方される患者像**
  - 小児喘息（5-11歳）：レルベア50
  - 成人喘息：症状に応じて100/200
  - アドヒアランス不良患者

#### 2.3 シムビコート（SMART療法の主役）
- **基本情報カード**
  - 成分：ICS/LABA（ホルモテロールの即効性）
  - 適応：喘息、COPD
  - SMART療法の独自性
- **タービュヘイラーの特徴**
  - 回転操作の重要性
  - 残量確認方法（赤マーク）
  - 垂直保持の必要性
- **SMART療法詳解**
  - 維持＋頓用の考え方
  - 用量設定（1日最大12吸入まで）
  - 適応患者の選択
- **処方される患者像**
  - 発作頻度の多い喘息患者
  - 用量調節が必要な患者

#### 2.4 メプチン（発作時の救世主）
- **基本情報カード**
  - 成分：SABA（プロカテロール）
  - 適応：喘息発作、COPD急性増悪
  - 3つのデバイス選択肢
- **デバイス別特徴**
  - エアー（pMDI）：同調の重要性
  - スイングヘラー：コンパクト設計
  - クリックヘラー：音で確認
- **SABA過剰使用問題**
  - 年間2本以上のリスク
  - モニタリングの重要性
- **処方される患者像**
  - すべての喘息患者（レスキュー）
  - 高齢者（ネブライザー使用）

### 4. 吸入指導実践ガイド（20%）

#### 4.1 デバイス別吸入手技
- **共通の基本手技**
  1. 息を吐き切る
  2. デバイス準備
  3. 吸入（デバイスにより速度が異なる）
  4. 息止め（5-10秒）
  5. うがい（ICS含有製剤）

- **デバイス特有の注意点**
  - エリプタ：30秒以内に吸入
  - タービュヘイラー：垂直保持
  - pMDI：同調の練習

#### 4.2 患者指導のコツ
- **小児への指導**
  - 遊び感覚での練習
  - 保護者への説明
- **高齢者への指導**
  - 吸入力の確認
  - スペーサーの活用
- **認知機能低下患者**
  - 簡単なデバイス選択
  - 家族・介護者への指導

#### 4.3 よくある失敗と対処法
- 吸入前に息を吐いていない
- デバイスに向かって息を吐く
- 吸入速度の誤り
- 息止めが不十分

### 5. 実践的ケーススタディ（10%）

#### ケース1：75歳男性、重症COPD
- 現在の治療：LAMA単剤
- 問題：増悪を繰り返す
- 提案：テリルジー100への変更
- 理由：トリプル療法、1日1回で簡便

#### ケース2：8歳女児、中等症喘息
- 現在の治療：ICS単剤、メプチン頻回使用
- 問題：コントロール不良
- 提案：レルベア50への変更
- 理由：1日1回でアドヒアランス向上

#### ケース3：45歳女性、喘息
- 現在の治療：ICS/LABA定期、発作時SABA
- 問題：月2-3回の発作
- 提案：シムビコートSMART療法
- 理由：維持＋頓用で発作予防

### 6. まとめとチェックリスト（5%）
- **吸入薬選択のポイント**
  - 疾患（喘息 vs COPD）
  - 重症度
  - 年齢・吸入力
  - アドヒアランス
  - デバイス操作能力

- **薬剤師チェックリスト**
  □ 適応症と用量の確認
  □ デバイス操作の実演
  □ 吸入手技の確認
  □ うがいの指導（ICS含有）
  □ 次回受診時の手技再確認
