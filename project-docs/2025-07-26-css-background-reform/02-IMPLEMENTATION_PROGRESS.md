## 実装の特徴
- **大きな領域**を白背景に統一
- **左ボーダー**でレベルや種類を色分け
- **ボックスシャドウ**で立体感を演出
- 小さなボックス（info-box等）の背景色は維持

## 効果
- 視覚的な余白が生まれ、読みやすさが向上
- 情報の階層が明確になった
- ページ全体の「色まみれ」感が解消
- レベルごとの識別は左ボーダーの色で可能
