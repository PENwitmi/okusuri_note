# 吸入薬4種類の調査結果

## 対象薬剤
1. **テリルジー（Trelegy）**
2. **レルベア（Relvar）**  
3. **シムビコート（Symbicort）**
4. **メプチン（Meptin）**

## 薬剤詳細情報

### 1. テリルジー・エリプタ（Trelegy Ellipta）
- **一般名**: フルチカゾンフランカルボン酸エステル/ウメクリジニウム臭化物/ビランテロールトリフェニル酢酸塩
- **成分構成**: ICS + LAMA + LABA（トリプル製剤）
- **適応**: COPD（慢性閉塞性肺疾患）
- **デバイスタイプ**: エリプタ（DPI）
- **用法**: 1日1回1吸入
- **特徴**: 
  - 世界初のトリプル配合吸入薬
  - 1日1回投与で利便性が高い
  - 重症COPD患者に適応

### 2. レルベア・エリプタ（Relvar Ellipta）
- **一般名**: フルチカゾンフランカルボン酸エステル/ビランテロールトリフェニル酢酸塩
- **成分構成**: ICS + LABA
- **適応**: 
  - 気管支喘息
  - COPD
- **デバイスタイプ**: エリプタ（DPI）
- **用法**: 1日1回1吸入
- **規格**: 
  - レルベア100（喘息）
  - レルベア200（喘息・COPD）
- **特徴**:
  - 1日1回投与で高いアドヒアランス
  - エリプタデバイスで吸入が簡単

### 3. シムビコート・タービュヘイラー（Symbicort Turbuhaler）
- **一般名**: ブデソニド/ホルモテロールフマル酸塩水和物
- **成分構成**: ICS + LABA
- **適応**: 
  - 気管支喘息
  - COPD
- **デバイスタイプ**: タービュヘイラー（DPI）
- **用法**: 
  - 維持療法：1回1-2吸入、1日2回
  - SMART療法：維持＋頓用使用可能
- **特徴**:
  - SMART療法が可能（維持＋頓用）
  - ホルモテロールの速効性
  - 用量調節が柔軟

### 4. メプチン（Meptin）
- **一般名**: プロカテロール塩酸塩水和物
- **成分構成**: SABA（短時間作用性β2刺激薬）
- **適応**: 
  - 気管支喘息の発作緩解
  - 慢性気管支炎、肺気腫の諸症状緩解
- **剤形**: 
  - メプチンエアー（pMDI）
  - メプチンスイングヘラー（DPI）
  - メプチンクリックヘラー（DPI）
- **用法**: 発作時に1-2吸入
- **特徴**:
  - 即効性（5-10分で効果発現）
  - 発作時のレスキュー薬
  - 多様なデバイス選択肢

## デバイス特性比較

### エリプタ（テリルジー、レルベア）
- **長所**: 
  - 1日1回で簡便
  - 吸入確認窓付き
  - 操作が簡単（カバーを開けて吸入）
- **短所**: 
  - 残量確認がカウンターのみ
  - 湿気に注意必要

### タービュヘイラー（シムビコート）
- **長所**: 
  - 用量調節可能
  - 残量確認窓あり
  - SMART療法対応
- **短所**: 
  - 1日2回投与
  - 吸入時の回転操作必要

### pMDI/DPI（メプチン）
- **長所**: 
  - 即効性
  - 携帯性良好
  - 複数デバイス選択可
- **短所**: 
  - pMDIは同調が必要
  - 頻回使用は喘息コントロール不良の指標

## 臨床での使い分け

### 重症度別選択
1. **軽症喘息**: メプチン頓用のみ
2. **中等症喘息**: レルベアまたはシムビコート定期使用
3. **重症喘息**: 高用量ICS/LABA（レルベア200、シムビコート高用量）
4. **重症COPD**: テリルジー（トリプル療法）

### 患者特性別選択
- **高齢者**: エリプタ（操作簡便）
- **若年者**: タービュヘイラー（用量調節可）
- **アドヒアランス不良**: 1日1回製剤（テリルジー、レルベア）
- **発作頻度高い**: シムビコート（SMART療法）＋メプチン

## 吸入指導のポイント

### 共通事項
1. 息を吐き切ってから吸入
2. 深く速く（DPI）または深くゆっくり（pMDI）吸入
3. 吸入後5-10秒息止め
4. ICS含有製剤は吸入後うがい必須

### デバイス別注意点
- **エリプタ**: カバーを開けた後は30秒以内に吸入
- **タービュヘイラー**: 垂直に立てて回転操作
- **pMDI**: スペーサー使用推奨（特に高齢者・小児）
