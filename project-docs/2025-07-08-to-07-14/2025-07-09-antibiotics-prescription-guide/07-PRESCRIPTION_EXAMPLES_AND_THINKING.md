# 実際の処方例と抗生物質選択の思考プロセス

## ケーススタディ：外来編

### Case 1: 28歳女性、排尿時痛と頻尿

**主訴**: 「3日前から排尿時に痛みがあり、トイレが近い」

**現病歴**
- 3日前から排尿時痛、頻尿（1時間毎）
- 残尿感あり、血尿なし
- 発熱なし、腰痛なし
- 性交渉歴あり（1週間前）

**既往歴**: 1年前にも同様の症状（膀胱炎）

**診察所見**
- 体温 36.8℃、血圧 110/70
- CVA叩打痛なし
- 下腹部軽度圧痛

**検査**
- 尿定性：白血球(3+)、亜硝酸塩(+)、潜血(±)
- 尿沈渣：WBC >50/hpf、細菌(+++)

#### 思考プロセス

1. **診断**: 単純性膀胱炎（若年女性、典型的症状）
2. **想定菌**: E.coli（80%）、その他グラム陰性桿菌
3. **重症度**: 軽症（発熱なし、腎盂腎炎示唆なし）
4. **耐性考慮**: キノロン耐性E.coli増加（20-30%）

#### 処方決定

```
Rp) フロモックス錠 100mg
    1回1錠（1日3錠）
    1日3回 毎食後 3日分
    
Rp) ロキソニン錠 60mg
    疼痛時 1回1錠
    5回分
```

**選択理由**
- セフカペン：単純性膀胱炎の第一選択
- 3日間：ガイドライン推奨の最短期間
- NSAIDs：症状緩和目的

**患者説明**
「細菌による膀胱炎です。3日間きちんと飲み切ってください。2-3日で症状は改善しますが、薬は最後まで飲んでください。症状が悪化したり、発熱したりしたらすぐ受診してください。」

### Case 2: 5歳男児、発熱と咽頭痛

**主訴**: 「昨日から39度の熱と喉の痛み」

**現病歴**
- 昨日朝から39.2℃の発熱
- 咽頭痛強く、食事摂取困難
- 咳なし、鼻汁なし
- 保育園で溶連菌流行中

**診察所見**
- 体温 39.0℃
- 咽頭発赤著明、白苔付着
- 両側扁桃腫大
- 前頚部リンパ節腫脹・圧痛
- 皮疹なし

**迅速検査**
- 溶連菌迅速抗原：陽性

#### 思考プロセス

1. **診断**: A群β溶連菌性咽頭炎（迅速検査陽性）
2. **合併症リスク**: リウマチ熱、糸球体腎炎
3. **治療目標**: 除菌（10日間が必須）
4. **体重**: 20kg

#### 処方決定

```
Rp) サワシリン細粒10%
    1回5g（1日15g）
    1日3回 毎食後 10日分
    
Rp) カロナール細粒20%
    1回1.5g（解熱鎮痛時）
    1日3回まで 10回分
```

**選択理由**
- アモキシシリン：溶連菌の第一選択
- 50mg/kg/日：小児の推奨用量
- 10日間：リウマチ熱予防に必須

**保護者説明**
「溶連菌による扁桃炎です。きちんと10日間飲まないと、後で心臓や腎臓の病気になることがあります。熱が下がっても必ず最後まで飲ませてください。」

### Case 3: 72歳男性、右下腿の発赤腫脹

**主訴**: 「3日前から右足が赤く腫れて痛い」

**現病歴**
- 3日前、右下腿に虫刺され様の発赤
- 翌日から急速に拡大、熱感・疼痛
- 昨日から38℃の発熱
- 足趾間の白癬あり

**既往歴**
- 2型糖尿病（HbA1c 8.2%）
- 高血圧
- 脂質異常症

**診察所見**
- 体温 38.2℃、血圧 140/80
- 右下腿に10×15cmの発赤、腫脹、熱感
- 境界やや不明瞭
- 圧痛著明
- 足趾間に浸軟・落屑

**検査**
- WBC 13,500、CRP 8.5

#### 思考プロセス

1. **診断**: 蜂窩織炎（典型的所見）
2. **侵入門戸**: 足白癬からの細菌侵入
3. **想定菌**: β溶連菌、黄色ブドウ球菌
4. **重症度**: 中等症（発熱、炎症反応上昇、糖尿病）
5. **MRSA リスク**: 低い（市中発症、初発）

#### 処方決定

```
Rp) ケフラール カプセル 250mg
    1回2カプセル（1日6カプセル）
    1日3回 毎食後 7日分
    
Rp) ロキソニン錠 60mg
    1回1錠（1日3錠）
    1日3回 毎食後 5日分
    
Rp) ムコスタ錠 100mg
    1回1錠（1日3錠）
    1日3回 毎食後 5日分
```

**選択理由**
- セファクロル：蜂窩織炎の第一選択
- 500mg×3回：中等症への適切な用量
- 7日間：標準的治療期間

**患者説明**
「細菌による皮膚の感染症です。抗生物質を1週間飲んでください。足を高くして安静にし、3日後に必ず再診してください。赤みが広がったり、熱が上がったらすぐ来てください。」

## ケーススタディ：入院編

### Case 4: 82歳女性、誤嚥性肺炎

**入院時現症**
- 特養入所中、認知症
- 食事中にむせ込み、その後発熱
- 意識レベル JCS I-3
- 体温 38.8℃、血圧 100/60、SpO2 88%（室内気）

**検査所見**
- WBC 15,200、CRP 18.5
- 胸部X線：右下肺野に浸潤影
- 喀痰培養：提出（結果待ち）

#### 思考プロセス

1. **診断**: 誤嚥性肺炎（確実）
2. **想定菌**: 口腔内常在菌（嫌気性菌含む）
3. **重症度**: 中等症（A-DROP 2点）
4. **初期治療**: 嫌気性菌カバー必須

#### 処方決定

```
入院時指示：
1) スルバシリン静注用 1.5g
   1回1.5g + 生食 100mL
   1日3回 8時、16時、24時 点滴静注
   
2) 補液：ソルデム3A 500mL×3本/日

3) 酸素投与：鼻カヌラ 2L/分
```

**48時間後の再評価**
- 解熱傾向、CRP低下
- 喀痰培養：通常の口腔内細菌叢
- 経口摂取再開可能

**処方変更**
```
Rp) オーグメンチン配合錠 250RS
    1回1錠（1日3錠）
    1日3回 毎食後 5日分
```

### Case 5: 35歳女性、複雑性腎盂腎炎

**主訴**: 「3日前から高熱と右腰痛」

**現病歴**
- 1週間前から膀胱炎症状
- 市販薬で様子見ていた
- 3日前から39℃台の発熱、右腰痛

**検査所見**
- WBC 18,500、CRP 22.3
- 尿培養：E.coli 10^5 CFU/mL
- 血液培養：2セット提出
- 腹部CT：右水腎症軽度

#### 思考プロセス

1. **診断**: 右腎盂腎炎＋軽度水腎症
2. **菌血症リスク**: 高い
3. **ESBL リスク**: 中等度（キノロン使用歴なし）
4. **初期治療**: 広域カバー

#### 初期治療

```
入院時指示：
1) ロセフィン静注用 1g
   1回2g + 生食 100mL
   1日1回 点滴静注
   
2) 補液：ラクテック 500mL×4本/日
```

**Day 3 培養結果**
- E.coli（ESBL陰性）
- セフトリアキソン感受性(S)

**De-escalation継続**
- ロセフィン継続 計7日間
- 解熱後、経口薬へスイッチ

```
退院処方：
Rp) クラビット錠 500mg
    1回1錠（1日1錠）
    1日1回 朝食後 7日分
```

## 抗生物質選択の思考フローチャート

### Step 1: 感染症の診断
```
症状・所見
    ↓
細菌感染症か？ → No → 抗生物質不要
    ↓ Yes
感染部位の特定
    ↓
重症度評価
```

### Step 2: 原因菌の推定
```
感染部位
    ↓
疫学的に頻度の高い菌
    ↓
患者背景（年齢、基礎疾患）
    ↓
耐性菌リスク評価
```

### Step 3: 抗菌薬選択
```
推定菌のカバー
    ↓
最も狭域なスペクトラム
    ↓
組織移行性
    ↓
副作用・相互作用
    ↓
コスト
```

### Step 4: 投与設計
```
標準用量の確認
    ↓
腎機能評価 → 要調整
    ↓
投与経路（経口 vs 注射）
    ↓
投与期間の設定
```

### Step 5: 効果判定と修正
```
48-72時間後評価
    ↓
改善あり → 継続 or De-escalation
    ↓
改善なし → 原因再検討
         → 耐性菌？
         → 膿瘍形成？
         → 非感染性？
```

## 処方時のピットフォール

### 避けるべき処方

1. **「念のため」の抗生物質**
   - ウイルス性上気道炎への処方
   - 明確な感染徴候のない発熱

2. **不適切な広域抗菌薬**
   - 単純性膀胱炎にキノロン系
   - 軽症皮膚感染にカルバペネム

3. **不十分な用量・期間**
   - 溶連菌咽頭炎で5日間のみ
   - 重症感染で低用量

4. **培養採取前の投与**
   - 重症例では培養後速やかに
   - 軽症では培養後の投与も可

### 処方の工夫

1. **服薬コンプライアンス向上**
   - 1日1回製剤の活用
   - 食事との関係が少ない薬剤

2. **副作用対策**
   - 下痢予防の整腸剤
   - マクロライドでの相互作用確認

3. **コスト考慮**
   - ジェネリック医薬品の活用
   - 必要最小限の投与期間
